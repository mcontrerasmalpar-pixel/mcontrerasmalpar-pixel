## 📄 Featured Research

### [QuechuaTok](https://github.com/mcontrerasmalpar-pixel/QuechuaTok) · *Published on arXiv*

Morphological Tokenization Benchmark for Southern Quechua — an agglutinative, low-resource language.

Compared BPE, Unigram LM, and WordPiece against a custom morphological segmenter (PRPE). Key finding: **BPE achieved only 6.67% morphological boundary accuracy**, while the rule-based segmenter reached **83.33%** — demonstrating that fertility rate alone is insufficient for evaluating tokenizers in languages with rich morphology.

📎 [arxiv.org/abs/2606.23943](https://arxiv.org/abs/2606.23943)

---

### [khipu-ml](https://github.com/mcontrerasmalpar-pixel/khipu-ml) · *Open Source*

Structural pattern mining on Inca khipus using unsupervised and supervised ML.

Applied **UMAP + HDBSCAN** for clustering and **XGBoost + SHAP** for classification on the Open Khipu Repository (Harvard). Results: 3 distinct clusters (silhouette = 0.769), F1 = 0.80 for Imperial Inca style detection, with "S-twist" identified as a key imperial signature (p = 3.7e-276).
